# Market Extension for MCEconomy

**Path:** `plugins/MCEconomy/extensions/Market`

This extension adds a fully functional, GUI-based shop system to the **MCEconomy** ecosystem. It bridges the gap between physical Minecraft items and digital economy currencies through an inventory interface.

## 🚀 Features
* **Infinite Markets:** Create unlimited categories (Ores, Food, Blocks).
* **In-Game Editor:** Drag-and-drop items to set up shops—no config editing required.
* **Asynchronous:** Database transactions happen off the main thread to prevent lag.
* **Base64 Storage:** Preserves every item detail (Enchants, NBT, Durability).

## 📖 Documentation
Full documentation, including command usage, configuration examples, and architecture details, can be found in the [`Web Documentation`](https://mcclauneck.github.io/market-website/).

## 📦 Build & Dependencies
* **Build System:** Gradle (Dynamic Versioning)
* **Dependencies:** MCExtension, MCEconomy-Common, Paper API
